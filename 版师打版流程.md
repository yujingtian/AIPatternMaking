# 女性弯腰头牛仔裤（贴合实操逻辑版）

> **坐标系设定（实操辅助线逻辑）**：
> *   **原点 (0, 0)**：设定为**外侧缝基础参考线**与**脚口水平线**的交点（即初始假设的脚口外缝参考点）。
> *   **Y轴正方向**：向上（朝向腰头方向），代表裤长。
> *   **X轴正方向**：向右（朝向内侧缝方向），代表裤宽。
> *   **单位统一**：厘米 (cm)。

### 初始尺寸参数参考
*   腰围：70，臀围：90，膝围：36，裤口：39
*   前浪：24，裤长：100，后浪：35

---

### 前片打版实操坐标化步骤

#### 1. 建立基础参考线与"大矩形"框架 (Reference Lines & Bounding Box)
起手先搭建外围框架。将外侧缝垂直参考线定为 `X = 0`。
*   **计算臀围宽度（确定内缝参考线）**：大概 21.7（计算：90/4 - 1前片差 + 0.2松量）。
    *   从外侧缝参考线向右移动 21.7，得到**内侧缝垂直参考线**：`X = 21.7`。
*   **绘制五条水平参考线**（在 X=0 到 X=21.7 之间拉横线）：
    *   **脚口参考线**：`Y = 0`。此时原点 `(0, 0)` 就是最初假设的外侧缝与脚口的交点。
    *   **膝围参考线**：`Y = 47.6`（立裆参考线 76.6 - 29 = 47.6）。
    *   **立裆参考线**：`Y = 76.6`（100.6 - 24前浪）。
    *   **臀围参考线**：`Y = 84.6`（立裆与腰围间距24的三等分处，即 76.6 + 8）。
    *   **腰围参考线**：`Y = 100.6`（100裤长 + 0.6松量）。
*   **完成大矩形**：此时构成了前片的基础网格。其中大矩形的四个角分别为：`(0, 0)`、`(21.7, 0)`、`(21.7, 100.6)`、`(0, 100.6)`。

#### 2. 确定立裆宽 (Crotch Extension)
立裆需要向内侧缝以外继续延伸。
*   **立裆宽**：臀围尺寸的0.04倍（约3.5cm）。
*   从大矩形的立裆内缝交点 `(21.7, 76.6)` 继续向右延伸 3.5cm。
*   **立裆宽顶点**：`(21.7 + 3.5, 76.6) = (25.2, 76.6)`。

#### 3. 细化绘制前浪基础线 (Detailed Front Rise Curve)
前浪是一条关键的"J"字形曲线，我们分上下两段并借助辅助点来精准绘制：
*   **第一步：计算下半段"J"底弧的角平分线辅助点**：
    *   内缝参考线（垂直）与立裆参考线（水平）的交叉点位于大矩形的内下角 `(21.7, 76.6)`。
    *   提取向上（内缝）和向右（立裆宽）的90度夹角，作45度的角平分线。
    *   沿此角平分线向右上延伸距离：`90(臀围) * 0.02 + 0.5 = 2.3 cm`。
    *   坐标换算（X与Y增量均为 2.3 * sin(45°) ≈ 1.63）：`X = 21.7 + 1.63 = 23.33`，`Y = 76.6 + 1.63 = 78.23`。
    *   **前浪下半段辅助点**：`(23.33, 78.23)`。
*   **第二步：绘制下半段凹弧**：
    *   将 **臀围内缝顶点 `(21.7, 84.6)`**、**前浪辅助点 `(23.33, 78.23)`**、**立裆宽顶点 `(25.2, 76.6)`** 这三点依次连接，画出一段向内凹的样条曲线。
*   **第三步：绘制上半段直线并整体圆顺**：
    *   **新腰围内缝参考顶点**：矩形的腰围内缝顶点 `(21.7, 100.6)` 沿腰围线向内（X轴负方向）收1cm，变为 `(20.7, 100.6)`。
    *   将 **新腰围内缝参考顶点 `(20.7, 100.6)`** 与 **臀围内缝顶点 `(21.7, 84.6)`** 相连，画一条直线。
    *   **整体圆顺**：确保上、下两段弧线在臀围内缝顶点 `(21.7, 84.6)` 处无缝、顺滑过渡，构成完整的前浪线。

#### 4. 确定重心基石：裤中线 (Center Crease Line)
这是实操中最核心的一步，腿部所有宽度都将以此线为对称轴分配，而非外侧缝参考线。
*   **裤中线位置**：取立裆总宽度（从 X=0 到 X=25.2）的中点，并向外缝微移。
    *   计算：`(0 + 25.2) / 2 = 12.6`。向外（X负方向）偏移0.6cm。
    *   **裤中线横坐标**：`X = 12.6 - 0.6 = 12.0`。以 `X = 12.0` 画一条垂直基准线。

#### 5. 计算并定位实际膝围与脚口顶点 (Actual Knee & Hem Width)
*   **膝围宽度**：前片单侧宽度为 8.4（计算：36/2 - 0.6*2）。
    *   **实际膝围内缝顶点**：`(12.0 + 8.4, 47.6) = (20.4, 47.6)`。
    *   **实际膝围外缝顶点**：`(12.0 - 8.4, 47.6) = (3.6, 47.6)`。
*   **脚口宽度**：前片单侧宽度为 9.15（计算：39/2 - 0.6*2）。
    *   **实际脚口内缝顶点**：`(12.0 + 9.15, 0) = (21.15, 0)`。
    *   **实际脚口外缝顶点**：`(12.0 - 9.15, 0) = (2.85, 0)`。

#### 6. 连接内侧缝与下部外侧缝 (Inner Seam & Lower Outer Seam)
*   **下部外侧缝**：实际脚口外缝顶点 `(2.85, 0)` 和 实际膝围外缝顶点 `(3.6, 47.6)`两点微微向内凹的画圆顺弧线。
*   **内侧缝线**：连接实际脚口内缝顶点 `(21.15, 0)`、实际膝围内缝顶点 `(20.4, 47.6)` 以及立裆宽顶点 `(25.2, 76.6)`。将三点连并画内收的圆顺弧线（脚口到膝围近乎直线，立裆到膝围的弧线要在膝围处与直线段平滑相切）。

#### 7. 完善上部外侧缝与最终腰线 (Upper Outer Seam & Final Waist Line)
*   **定位最终腰围外缝顶点**：
    *   腰围所需宽度：18.4cm（70腰围/4 + 0.3松量 + 0.6省道）。
    *   从新腰围内缝参考顶点 `(20.7, 100.6)` 向外（X负方向）沿腰围线测 18.4cm：`20.7 - 18.4 = 2.3`。
    *   **最终腰围外缝顶点**：`(2.3, 100.6)`。
*   **上部外侧缝**：将实际膝围外缝顶点 `(3.6, 47.6)`、矩形上的臀围外缝参考顶点 `(0, 84.6)`、最终腰围外缝顶点 `(2.3, 100.6)` 三点用向外凸出的圆顺弧线连接。
*   **定位最终腰围内缝顶点（定前浪长）**：
    *   前浪线总长需严格等于 24cm。
    *   从立裆宽顶点 `(25.2, 76.6)` 顺着之前画好的前浪基础曲线往上测量，切断在长度正好为 24cm 的位置。这个点即为 **最终腰围内缝顶点** `(X_内, Y_内)`。
*   **绘制弯腰头最终腰围线**：
    *   取 **最终腰围外缝顶点 `(2.3, 100.6)`** 和 **最终腰围内缝顶点 `(X_内, Y_内)`** 连线的中点作为贝塞尔控制点。
    *   将该控制点的Y坐标向下（朝臀围线方向）微微移动 0.4cm。
    *   以上述三点（外缝顶点、下移的控制点、内缝顶点）绘制贝塞尔曲线，完成腰围线。

#### 8. 补充下腰头
* 上腰头：腰围外缝顶点和腰围内缝顶点之间的腰围线
* 下腰头外端点：从腰围外缝顶点顺着外侧缝曲线向下4cm
* 下腰头内端点：从腰围内缝顶点顺着前浪曲线向下4cm
* 下腰头弧线：连接下腰头外端点和下腰头内端点，弧度与上腰头一致（控制点下移量相同）

#### 9. 绘制前门襟
* 从下腰头内端点顺着下腰头找3cm叫下腰头门襟点，画一条直线与前浪平行的线叫门襟线，画到和臀围线交叉再延伸1cm的点叫门襟内端点，然后再通过门襟内端点垂直于前浪画一条与前浪交叉的线，这个线叫门襟宽线，交叉点叫门襟外端点，定好一个基本框架
* 从下腰头门襟点顺着门襟线画线，画到离门襟内端点3公分的位置就相切地出去画弧线，最后相切于门襟宽线

#### 10. 绘制月牙袋
* **月牙袋外缝顶点**：从下腰头外缝顶点沿着外侧缝曲线向下7cm取点，坐标记为 `(x_pocket_outer, y_pocket_outer)`
* **月牙袋宽顶点**：从下腰头外缝顶点开始沿着下腰头曲线向内侧方向量取10cm取点，坐标记为 `(x_pocket_width, y_pocket_width)`
* **月牙袋弧线**：用三次贝塞尔曲线将两个顶点连接，弧线向X正方向和Y负方向所指的区域凸起（即向右下方凸起）。使用两个控制点来控制弧度：
  - 控制点1：从月牙袋外缝顶点向右下方偏移约3cm
  - 控制点2：从月牙袋宽顶点向右下方偏移约3cm
* **月牙袋闭合区域**：
  - 由三个点围成：月牙袋外缝顶点、月牙袋宽顶点、上腰头外缝顶点
  - 其中月牙袋外缝顶点到月牙袋宽顶点用贝塞尔弧线连接
  - 月牙袋宽顶点到上腰头外缝顶点用下腰头弧线连接
  - 上腰头外缝顶点到月牙袋外缝顶点用外侧缝弧线连接

#### 11.绘制月牙袋的省道
* **月牙袋省道点**：从月牙袋宽顶点再沿着下腰头线向内侧方向延伸0.6cm，坐标记为 `(x_pocket_dart, y_pocket_dart)`
* **月牙袋省道弧线**：用贝塞尔曲线将月牙袋省道点和月牙袋外缝顶点连接，弧线向X正方向和Y负方向所指的区域凸起（即向右下方凸起）
* **月牙袋省的边线（宽顶点侧）**：从月牙袋宽顶点画一条垂直于上腰围线的直线，一直画到与上腰围线交叉为止
* **月牙袋省的边线（省道点侧）**：从月牙袋省道点画一条垂直于上腰围线的直线，一直画到与上腰围线交叉为止

#### 12.绘制袋贴
* **袋贴下腰头顶点**：从月牙袋省道点 `(x_pocket_dart, y_pocket_dart)` 沿着下腰头线继续向内侧方向扩展3.5cm取点，坐标记为 `(x_patch_waist, y_patch_waist)`
* **袋贴外缝顶点**：从月牙袋外缝顶点 `(x_pocket_outer, y_pocket_outer)` 沿着外侧缝曲线向Y负方向（向下）继续扩展3.5cm取点，坐标记为 `(x_patch_outer, y_patch_outer)`
* **袋贴弧线**：用三次贝塞尔曲线将袋贴下腰头顶点和袋贴外缝顶点连接，弧线始终与月牙袋省道弧线保持3.5cm的距离（即沿月牙袋弧线的外法线方向，向X正方向、Y负方向所指的区域偏移3.5cm）。两个控制点分别为月牙袋省道弧线两个控制点沿外法线方向偏移3.5cm后的点：
  - 控制点1（靠近袋贴下腰头顶点）：月牙袋省道弧线宽顶点侧的控制点沿外法线偏移3.5cm
  - 控制点2（靠近袋贴外缝顶点）：月牙袋省道弧线外缝顶点侧的控制点沿外法线偏移3.5cm
* **袋贴闭合区域**（袋贴贴片）：
  - 由四段围成：袋贴弧线、外侧缝弧线（袋贴外缝顶点 → 月牙袋外缝顶点）、月牙袋省道弧线（反向：月牙袋外缝顶点 → 月牙袋省道点）、下腰头弧线（月牙袋省道点 → 袋贴下腰头顶点）
  - 即月牙袋省道弧线与袋贴弧线之间约3.5cm宽的窄条区域

#### 13.绘制袋布
* **袋布上腰头顶点**：从袋贴下腰头顶点 `(x_patch_waist, y_patch_waist)` 沿着下腰头线继续向内侧方向延伸 0.5-0.8cm（取中约0.65cm）取点，坐标记为 `(x_bag_waist, y_bag_waist)`
* **袋布线**：从袋布上腰头顶点沿着垂直于下腰头线的方向、朝向立裆线（Y负方向）画一条直线，记为袋布线
* **袋布内端点**：袋布线一直画到离立裆参考线 `Y = 76.6` 仅 1cm 处（即 `Y = 77.6`）停下，坐标记为 `(x_bag_inner, y_bag_inner)`
* **袋布拐点**：从袋布内端点垂直于袋布线、朝向外缝（X负方向）画 5-6cm（取中约5.5cm）的直线，顶点坐标记为 `(x_bag_corner, y_bag_corner)`
* **袋布外缝顶点**：从袋贴外缝顶点 `(x_patch_outer, y_patch_outer)` 沿外侧缝曲线向Y负方向（向下）继续扩展 2-3cm（取中约2.5cm）取点，坐标记为 `(x_bag_outer, y_bag_outer)`
* **袋布弧线**：用三次贝塞尔曲线将袋布拐点和袋布外缝顶点连接，拐点处沿底边方向（朝外缝）相切，外缝顶点处沿外侧缝方向相切，形成圆顺弧线
* **袋布闭合区域**（袋布片）：
  - 由四段围成：袋布线（袋布上腰头顶点 → 袋布内端点）、底边（袋布内端点 → 袋布拐点）、袋布弧线（袋布拐点 → 袋布外缝顶点）、顶边（袋布外缝顶点沿外侧缝向上至下腰头外缝顶点，再沿下腰头至袋布上腰头顶点）

#### 14. 绘制小表袋
* **外缝上参考点**：从下腰头外缝顶点沿外侧缝曲线向Y负方向（向下）3-4cm（取中约3.5cm）取点，作为小表袋外线的上端高度参考
* **小表袋外线**：过外缝上参考点，沿外侧缝方向（平行于外缝线）、向裤身内侧（X正方向）偏移 3cm 画一条平行线，记为小表袋外线
* **小表袋上外端点**：小表袋外线离下腰头 3-4cm（取中约3.5cm）处的上端顶点，即外缝上参考点向内偏移3cm后的点，坐标记为 `(x_wp_outer_up, y_wp_outer_up)`
* **小表袋下外端点**：小表袋外线沿Y负方向一直画到与袋贴弧线交叉，交叉点坐标记为 `(x_wp_outer_down, y_wp_outer_down)`
* **小表袋上内端点**：从小表袋上外端点垂直于小表袋外线、向裤身内侧（X正方向）画 5cm 的垂直线，顶点坐标记为 `(x_wp_inner_up, y_wp_inner_up)`
* **小表袋下内端点**：从小表袋上内端点画一条平行于小表袋外线、方向为Y负方向的直线，一直画到与袋贴弧线交叉，交叉点坐标记为 `(x_wp_inner_down, y_wp_inner_down)`
* **小表袋闭合区域**（小表袋片）：
  - 由四段围成：顶边（小表袋上外端点 → 小表袋上内端点）、内线（小表袋上内端点 → 小表袋下内端点）、底边（沿袋贴弧线 小表袋下内端点 → 小表袋下外端点）、外线（小表袋下外端点 → 小表袋上外端点）


### 后片打版实操坐标化步骤
#### 1. 建立基础参考线与"大矩形"框架 (Reference Lines & Bounding Box)
起手先搭建外围框架。将外侧缝垂直参考线定为 `X = 0`。
*   **计算臀围宽度（确定内缝参考线）**：大概 23.7（计算：90/4 + 1前片差 + 0.2松量）。
    *   从外侧缝参考线向右移动 23.7，得到**内侧缝垂直参考线**：`X = 23.7`。
*   **绘制五条水平参考线**（在 X=0 到 X=23.7 之间拉横线）：
    *   **脚口参考线**：`Y = 0`。此时原点 `(0, 0)` 就是最初假设的外侧缝与脚口的交点。
    *   **膝围参考线**：`Y = 47.6`（立裆参考线 76.6 - 29 = 47.6）。
    *   **立裆参考线**：`Y = 76.6`（与前片共用同一直裆深度，即 100.6 - 24前浪）。
    *   **臀围参考线**：`Y = 84.6`（立裆与腰围间距24的三等分处，即 76.6 + 8）。
    *   **腰围参考线**：`Y = 100.6`（100裤长 + 0.6松量）。
*   **完成大矩形**：此时构成了后片的基础网格。其中大矩形的四个角分别为：`(0, 0)`、`(23.7, 0)`、`(23.7, 100.6)`、`(0, 100.6)`。

#### 2. 确定后腰头线宽度 (Back Waistband Width)
*   **后腰围外缝顶点**：从大矩形的腰围外缝参考顶点 `(0, 100.6)` 沿腰围线向内（X正方向）收 0.5cm，得到 `(0 + 0.5, 100.6) = (0.5, 100.6)`。
*   **后腰围内缝顶点**：从后腰围外缝顶点 `(0.5, 100.6)` 沿腰围线（X正方向）测量 19.8cm（计算：腰围 70/4 + 0.3松量 + 2省道 = 17.5 + 0.3 + 2 = 19.8），得到 `(0.5 + 19.8, 100.6) = (20.3, 100.6)`。

#### 3. 确定后立裆宽和落档 (Back Crotch Width & Crotch Drop)
*   **后立裆宽**：约 9cm（计算：臀围 90 × 0.1 = 9）。
*   **立裆延伸点**：从大矩形的立裆内缝交点 `(23.7, 76.6)` 沿立裆线（X正方向）延伸后立裆宽 9cm，得到 `(23.7 + 9, 76.6) = (32.7, 76.6)`。
*   **后立裆宽顶点（落档后）**：将立裆延伸点向 Y 负方向下落 1cm（落档量），得到 `(32.7, 76.6 - 1) = (32.7, 75.6)`。
*   **落档线**：过后立裆宽顶点 `(32.7, 75.6)` 作水平线（平行于臀围线），向左垂直相交于内侧缝参考线 `X = 23.7`，即从 `(23.7, 75.6)` 到 `(32.7, 75.6)` 的水平线段。

#### 4. 绘制后浪基础线 (Detailed Back Rise Curve)
后浪是一条整体圆顺的曲线，借助"困势线"和辅助点精准绘制（构造类比前浪）：
*   **第一步：求困势顶点 K（困势线与落档线的交点）**：
    *   **困势线**：过 **后腰围内缝顶点 `(20.3, 100.6)`** 与 **臀围内缝顶点 `(23.7, 84.6)`** 的直线。
    *   将困势线向 Y 负方向延长，与**落档线 `Y = 75.6`** 相交。
    *   计算：方向 `(3.4, -16)`，参数 `t = (75.6 - 100.6) / (-16) = 1.5625`，`X = 20.3 + 3.4 × 1.5625 = 25.61`。
    *   **困势顶点 K**：`(25.61, 75.6)`。
*   **第二步：求后浪辅助点 H（困势线与落档线的角平分线上）**：
    *   在困势顶点 K 处，取**困势线方向（指向后腰围内缝顶点，约 `(-0.207, 0.978)`）**与**落档线方向（指向后立裆宽顶点，`(1, 0)`）**两射线的角平分线，方向指向 X 正、Y 正（约 `(0.627, 0.779)`）。
    *   沿角平分线量取距离：`0.02 × 90(臀围) + 0.5 = 2.3 cm`。
    *   **辅助点 H**：`(25.61 + 2.3×0.627, 75.6 + 2.3×0.779) = (27.05, 77.39)`。
*   **第三步：绘制上半段（后腰围内缝顶点 → 臀围内缝顶点）**：从 `(20.3, 100.6)` 到 `(23.7, 84.6)`，画一段微微向 X 负方向凹的圆顺弧线。
*   **第四步：绘制下半段（臀围内缝顶点 → 辅助点 H → 后立裆宽顶点）**：从 `(23.7, 84.6)` 经过辅助点 `(27.05, 77.39)` 到后立裆宽顶点 `(32.7, 75.6)`，画一段向 X 负、Y 负方向凹的 J 型弧线。
*   **整体圆顺**：上、下两段在臀围内缝顶点 `(23.7, 84.6)` 处切线方向一致（G1 连续），合起来构成一条完整的后浪曲线。

#### 5. 绘制裤中线 (Center Crease Line)
腿部所有宽度将以裤中线为对称轴分配。
*   **取中点**：从**立裆外缝顶点 `(0, 76.6)`** 到**立裆延伸点 `(32.7, 76.6)`** 的立裆线段取中点：`(0 + 32.7) / 2 = 16.35`，即 `(16.35, 76.6)`。
*   **向外侧缝平移**：将该中点向**外侧缝参考线（X 负方向）平移 2.5cm**，得到裤中线基准横坐标 `16.35 - 2.5 = 13.85`。
*   **裤中线**：以 **`X = 13.85`** 画一条平行于内侧缝参考线（竖直）的基准线，从脚口参考线 `Y = 0` 一直连接到腰围参考线 `Y = 100.6`。

#### 6. 计算并定位实际膝围与脚口顶点 (Actual Knee & Hem Width)
以后片裤中线 `X = 13.85` 为对称轴分配膝围与脚口宽度（后片比前片宽，用 `+0.6*2`，与前片相反；前后片相加正好等于围度）。
*   **膝围宽度**：后片单侧宽度为 9.6（计算：`(36/2 + 0.6×2) / 2 = (18 + 1.2) / 2 = 9.6`）。
    *   **实际膝围外缝顶点**：从裤中线与膝围线交点 `(13.85, 47.6)` 向 X 负方向量 9.6cm，得到 `(13.85 - 9.6, 47.6) = (4.25, 47.6)`。
    *   **实际膝围内缝顶点**：向 X 正方向量 9.6cm，得到 `(13.85 + 9.6, 47.6) = (23.45, 47.6)`。
*   **脚口宽度**：后片单侧宽度为 10.35（计算：`(39/2 + 0.6×2) / 2 = (19.5 + 1.2) / 2 = 10.35`）。
    *   **实际脚口外缝顶点**：从裤中线与脚口线交点 `(13.85, 0)` 向 X 负方向量 10.35cm，得到 `(13.85 - 10.35, 0) = (3.5, 0)`。
    *   **实际脚口内缝顶点**：向 X 正方向量 10.35cm，得到 `(13.85 + 10.35, 0) = (24.2, 0)`。

#### 7. 绘制内缝和外缝 (Side Seam & Inseam)
以后片各关键顶点连线，分段画圆顺弧线，相邻段在交点处相切（G1 连续）。
*   **后臀围外缝顶点**：外侧缝参考线 `X = 0` 与臀围参考线 `Y = 84.6` 的交点，即 `(0, 84.6)`。

**外缝（外侧，从腰到脚口，3 段）：**
*   **上段（后腰围外缝顶点 → 后臀围外缝顶点）**：从 `(0.5, 100.6)` 到 `(0, 84.6)`，向**外（X 负方向）**圆顺凸出，相切于后臀围外缝顶点。
*   **中段（后臀围外缝顶点 → 后膝围外缝顶点）**：从 `(0, 84.6)` 到 `(4.25, 47.6)`，**先凸（X 负）后凹（X 正）**的 S 型弧线（带拐点），相切于后膝围外缝顶点。
*   **下段（后膝围外缝顶点 → 脚口外缝顶点）**：从 `(4.25, 47.6)` 到 `(3.5, 0)`，向**内（X 正方向）**微微凹，相切于脚口外缝顶点。

**内缝（内侧，从立裆到脚口，2 段）：**
*   **上段（后立裆宽顶点 → 后膝围内缝顶点）**：从 `(32.7, 75.6)` **先沿 Y 负、X 负方向的角平分线（45° 向左下）出发**，再逐渐圆顺过渡，**相切于后膝围内缝顶点 `(23.45, 47.6)`**（末端切线近竖直，与下段平滑连接）。
*   **下段（后膝围内缝顶点 → 脚口内缝顶点）**：从 `(23.45, 47.6)` 到 `(24.2, 0)`，向**内（X 负方向）**微微凹，相切于脚口内缝顶点。

#### 8. 绘制最终腰围线 (Final Waist Line)
*   **新腰围内缝顶点（定后浪长）**：从**后立裆宽顶点 `(32.7, 75.6)`** 沿后浪曲线向上量取后浪长 `35cm`。当前后浪基础曲线弧长约 `30cm`，不足部分 `(35 − 30) ≈ 5cm` 从曲线起点 `(20.3, 100.6)` **沿其切线向后上方延伸**，得到新腰围内缝顶点约 `(19.7, 105.6)`（即后翘，高于腰围参考线 `Y = 100.6`）。
*   **新腰围外缝顶点**：从新腰围内缝顶点作**后浪在该点的垂线**；同时将外缝上段弧（后臀围外缝顶点 `(0, 84.6)` → 后腰围外缝顶点 `(0.5, 100.6)`）从 `(0.5, 100.6)` 沿切线向上延伸。该垂线与延伸弧的交点即为新腰围外缝顶点。
*   **最终腰围线**：连接新腰围外缝顶点与新腰围内缝顶点。

#### 9. 绘制腰省 (Waist Dart)
后片腰部收一个省道（省宽共 2cm，对应步骤2 的"+2省道"）。
*   **省中点**：取最终腰围线（新腰围外缝顶点 `(1.42, 103.29)` → 新腰围内缝顶点 `(20.18, 105.60)`）的中点，约 `(10.80, 104.45)`。

*   **省外端点 / 省内端点**：从省中点沿腰围线向两侧各量 `1cm`，得到省外端点（靠外缝侧）约 `(9.81, 104.32)`、省内端点（靠内缝侧）约 `(11.79, 104.57)`。

#### 后腰裁片
* 和前腰头裁片一样需要把省部分去掉，然后把左边右边拼合而成，拼合之后会有一个明显的夹角，需要将中间部分画弧线，弧度是向内的，弧线会微微高于省内端点和外端点。以上腰头为例，弧线最终会在离上腰头内缝顶点和上腰头外缝顶点3cm的地方相切于上腰头线，也就是两边是直的，中间是弧线。
* 下腰头整体离上腰头4cm，保持这个间距，弧线路线是一致。

后腰裁片（后腰头）为独立裁片，由后片上腰头线与下腰头线之间的腰头条去掉 2cm 腰省后拼合而成。
*   **省道拼合（刚体变换）**：以**省尖**为旋转基点，将腰省内缝侧部分（上腰头内缝段 + 后浪侧边 + 下腰头内缝段）整体旋转，使上腰头的**省内端点**与**省外端点**重合（旋转角度取两段省边线方向之差，即省尖处夹角）。
*   **上腰头轮廓（两边直线 + 中间弧线）**：拼合后上腰头在省拼合点处形成一个明显的夹角，需用弧线圆顺：
    *   **外缝侧直线段**：从上腰头外缝顶点沿上腰头线取 3cm，为直线段。
    *   **内缝侧直线段**：从上腰头内缝顶点（拼合变换后）沿上腰头线取 3cm，为直线段。
    *   **中间弧线**：用三次贝塞尔曲线连接两个 3cm 直线段的端点，两个控制点同取两直线段所在直线的交点（即省拼合点），使弧线两端分别与两直线段精确相切；弧度向内（凹向腰头内部），弧顶微微高于拼合后的省内端点/外端点（约 0.1cm）。
*   **下腰头轮廓**：将上腰头轮廓（直线段 + 弧线）整体沿法线方向向腰头内部平行偏移 4cm，保持处处 4cm 间距，弧线路线与上腰头一致。
*   **轮廓构成（闭合，从上腰头外缝顶点出发）**：
    1.  **上腰头段**：上腰头外缝顶点 → 外缝侧直线段 → 中间弧线 → 内缝侧直线段 → 上腰头内缝顶点（拼合变换后）。
    2.  **内缝侧边**：上腰头内缝顶点 → 沿端点处法线向下 4cm → 下腰头内端点。
    3.  **下腰头段**：下腰头内端点 → 沿下腰头轮廓（反向）→ 下腰头外端点。
    4.  **外缝侧边**：下腰头外端点 → 沿端点处法线向上 4cm → 上腰头外缝顶点，闭合轮廓。
*   **单独绘制**：导出时将裁片整体向 Y 负方向平移到袋布裁片下方（与上方图形间隔 5cm）单独展示（SVG 中为深橙色闭合轮廓并标注"后腰头"，DXF 中为 `BACKWAISTBANDPANEL` 图层）。

#### 机头裁片
* 机头裁片和后腰头思路一样，省的部分要去掉，然后将左右两部分拼合，然后中间会有夹角，用弧线画圆顺

机头裁片（育克）为独立裁片，由下腰头线、机头线、外侧缝、后浪围成的机头区域去掉腰省后拼合而成。
*   **机头区域（拼合前）**：由四段围成——顶边（下腰头弧线：下腰头外端点 → 下腰头内端点）、内缝侧边（沿后浪：下腰头内端点 → 机头内缝顶点）、底边（机头线：机头内缝顶点 → 机头外缝顶点）、外缝侧边（沿外侧缝：机头外缝顶点 → 下腰头外端点）。
*   **腰省穿过机头**：腰省省尖位于机头线之下，两条省边线（省外端点 → 省尖、省内端点 → 省尖）穿透整个机头，分别与**下腰头线**交于上省交点（外/内）、与**机头线**交于下省交点（外/内）；机头内的省区为一个梯形（上宽下窄），需整体挖除。
*   **省道拼合（刚体变换）**：将内缝侧部分（上腰头内缝段 + 后浪侧边 + 机头线内缝段）整体旋转平移，使**上省交点（内）与上省交点（外）重合**、省边线（内）方向与省边线（外）方向一致（下省交点处的间距残差小于 0.1cm，在圆顺时自然吸收）。
*   **拼合后圆顺**：拼合处顶边（下腰头线）与底边（机头线）各形成一个夹角，类比后腰头的"两边直线 + 中间弧线"方式分别画圆顺：
    *   顶边与底边各自从两端点起保留 3cm 直线段，中间部分整体用一条三次贝塞尔弧线过渡；
    *   弧线的两个控制点同取两侧边缘所在直线的交点（即拼合夹角点附近），使弧线两端与直线段精确相切，把夹角的方向差分摊到整条弧线上，圆顺过渡。
*   **轮廓构成（闭合，从下腰头外端点出发）**：
    1.  **顶边（外缝侧）**：下腰头外端点 → 沿下腰头线 → 上省交点（外），经圆顺弧线；
    2.  **顶边（内缝侧，拼合后）**：上省交点（内）→ 沿下腰头线 → 下腰头内端点（经刚体变换），经圆顺弧线；
    3.  **内缝侧边**：下腰头内端点 → 沿后浪 → 机头内缝顶点（经刚体变换）；
    4.  **底边（内缝侧，拼合后）**：机头内缝顶点 → 沿机头线 → 下省交点（内）（经刚体变换），经圆顺弧线；
    5.  **底边（外缝侧）**：下省交点（外）→ 沿机头线 → 机头外缝顶点，经圆顺弧线；
    6.  **外缝侧边**：机头外缝顶点 → 沿外侧缝向上 → 下腰头外端点，闭合轮廓。
*   **单独绘制**：导出时将裁片整体向 Y 负方向平移到后腰头裁片下方（与上方图形间隔 5cm）单独展示（SVG 中为紫色闭合轮廓并标注"机头"，DXF 中为 `JITOUPANEL` 图层）。
